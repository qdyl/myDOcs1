              # window 快捷键

              快捷键说明
              Windows徽标键 + R打开“运行”命令
              Windows徽标键 + E打开“文件资源管理器”
              Windows徽标键 + I打开“设置”
              Windows徽标键 + D隐藏或取消隐藏桌面
              Windows徽标键 + 向上箭头焦点窗口最大化
              Windows徽标键 + 向下箭头焦点窗口最小化
              Windows徽标键 + M最小化所有窗口
              Alt + Tab打开“任务切换程序”，每按一次 Tab 即向前移动一次，松开后切换到该窗口
              Shift + Alt + Tab打开“任务切换程序”，每按一次 Tab 即向后移动一次，松开后切换到该窗口

              # 终端命令行

              技巧提示：
              1、在文件夹地址栏中输入 cmd，按 Enter键， 即可在当前目录下打开 cmd 终端。
              2、或 按住 Shift 键 ，然后点击鼠标右键，可在当前目录下打开 cmd 终端。

              # 文件（夹）操作类

              说明终端命令
              创建文件夹目录md 或 mkdir
              删除文件夹目录rm 或 rmdir
              创建空文件type nul > test.txt 或 type > test.txt
              输入内容到文件echo 内容 > test.txt
              复制文件copy
              移动（剪切）文件move a.txt d:\b.txt
              删除文件del
              重命名文件ren
              显示文本文件的内容type
